### To run the app: 
```js
git clone QUANTUM-SOCIAL
cd quantum-social
npm install
npm run dev

// Server is runnung on "http://localhost:5173"
```

>  **Note:** 'local.env' file with necessary app keys is needed in order to connect to Backend as a Service **"APPWRITE"**!
